# React + GSAP Animation Project

## Overview
This project is a React application built with Vite that demonstrates powerful animations using GSAP and the [@gsap/react](https://greensock.com/react/) integration. The application features dynamic SVG animations, interactive transitions, and mousemove parallax effects to create an engaging user experience.

## Features
- **React:** For component-based UI development.
- **Vite:** Fast build times and hot module replacement.
- **GSAP:** Smooth, high-performance animations.
- **Tailwind CSS:** Utility-first styling framework.
- **Interactive Animations:** SVG mask animations, scaling, rotation, and parallax effects.

## Installation

1. **Clone the repository:**
    ```sh
    git clone <repository-url>
    ```
2. **Navigate to the project directory:**
    ```sh
    cd animation-react
    ```
3. **Install dependencies:**
    ```sh
    npm install
    ```

## Development Server

To start the development server, run:

```sh
npm run dev
